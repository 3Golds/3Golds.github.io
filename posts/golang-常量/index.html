Golang 常量

2017-12-14 — Written by zhaoxin — 1 min read
#go 

在 Go 语言中，常量是指编译期间就已知且不可改变的值。常量可以是数值类型(包括整数、浮点数和负数类型)、布尔类型、字符串类型等。

常量定义

通过 const 关键字定义，有以下三种方式


const v int = 8989

const fu = 0.2

const (

    long int64 = 2048

    size = -1

)

预定义常量

Go 语言预定义了一些常量：true、false 和 iota。其中 iota 比较特殊，可以被认为是一个可被编译器修改的常量，在第一个 const 关键字出现时被重置为 0，然后在下一个 const 出现之前，每出现一次 iota，其所代表的数字会自动增 1，我们看一个例子。


const (

	c0 = iota    // iota被重设为0

	c1 = iota    // c1 = 1

	c2           // c2 = 2

)
